## Introduction                             {#sct_mainpage_introduction}

This is the documentation for the **internal codebase** of the Thallium project. Content on this web page is mostly intended for **development** of
the Thallium library - as the documented functions are used internally - rather than for **using** the library. For documentation on the **public
interface** of the project, please see the <a href="../index.html">Public API Homepage</a>.
