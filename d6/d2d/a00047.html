unsigned int *const  out_count,
const char **  out_layer_names 
)

This functions gets an array of required layers for Vulkan instances, from the information provided through the given config structure.

These layers are enabled on all Vulkan instances created with (internal function) thvk_CreateInstance().

Parameters
configPointer to a Vulkan renderer configuration
out_countlocation to which the amount of layers will be returned
out_layer_namesNULL or the array to output layer names into.
Returns
Return status of the function. 0 indicates an error, and 1 indicates success.

◆ thvk_GetRequiredDeviceExtensions()

int thvk_GetRequiredDeviceExtensions ( unsigned int *const  out_count,
const char **const  out_extension_names 
)

This functions populates an array of required extensions for Vulkan physical devices.

The suitability of physical devices is partially based on these extensions. If a device does not support all of the required extensions, it will not be considered suitable for the program.

Parameters
out_countlocation to which the amount of extensions will be returned
out_extension_namesNULL or the array to output extension names into.
Returns
Array of extension names (NULL if error)
