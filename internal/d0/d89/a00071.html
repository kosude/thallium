Detailed Description

Some internally functions for displaying information about Vulkan structures.

Normally, the information must be supplied - these functions just provide ways of printing them as formatted strings.

Note
The majority of the functions in this group only have any effect if THALLIUM_DEBUG_LAYER is enabled due to the high frequency of heavy string operations.
