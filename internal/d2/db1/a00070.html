int thvk_CreateInstance ( thvk_RenderSystem_t *  render_system)

This function creates a Vulkan instance into the render system render_system.

If render_system includes a debugger, a debug messenger will also be created for it.

Parameters
render_systemPointer to the render system into which the instance will be returned.
Returns
Return status of the function. 0 indicates an error, and 1 indicates success.
