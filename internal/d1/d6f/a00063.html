const th_RendererConfig_Vulkan_t *  config

This function creates a Vulkan instance into the location instance.

If debugger is not NULL, a debug messenger will be created alongside the instance for Vulkan debugging.

Parameters
renderSystemPointer to the render system into which the instance will be returned.
configPointer to a Vulkan renderer configuration structure.
Returns
Return status of the function. 0 indicates an error, and 1 indicates success.
