Vulkan device managers

This section documents the device manager object found in Vulkan renderer systems, and their associated functions.

Types

Objects

typedef struct TLVK_DeviceManager_t TLVK_DeviceManager_t

An object to store handles to a Vulkan physical and logical device.

A Thallium Vulkan device manager stores handles to a Vulkan physical and logical device for use in Vulkan renderer systems.

See also

TLVK_DeviceManagerCreate()

Descriptors

struct TLVK_DeviceManagerDescriptor_t

Descriptor struct to configure the creation of a Thallium Vulkan device manager.

This descriptor structure provides options for the creation of Thallium device manager for Vulkan.

Functions

TLVK_DeviceManager_t *TLVK_DeviceManagerCreate(const TLVK_RendererSystem_t *const renderer_system, const TLVK_DeviceManagerDescriptor_t descriptor)

Create and return a heap-allocated Vulkan device manager.

This function creates a new Vulkan device manager, choosing and interfacing with GPU device(s) based on configuration given in descriptor. If there were any errors during creation, NULL will be returned instead of the device manager object.
