Vulkan Module

Objects and functions defined within the Thallium Vulkan module (compiled using -DTHALLIUM_BUILD_MODULE_VULKAN=ON).

Note

These Vulkan-specific abstractions are not exposed by default. You will need to #include the additional thallium_vulkan.h file in order to access them.
