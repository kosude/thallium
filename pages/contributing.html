Contributing

Building For Development

When building the project for development, make sure to use the -DCMAKE_BUILD_TYPE=Debug flag. It is also recommended to compile test executables in this case.

Adding To The Documentation

Document all new public functions or types with Doxygen-style formatting. Also see the Breathe documentation to find out how to include your code’s documentation in the HTML output.

Resources

I keep an upstream TO-DO list on Trello. Submit issues and pull requests to the GitHub repo.
