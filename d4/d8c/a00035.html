Validate the given instance extension.

If layerName is not NULL, that layer is used to validate the extension. Otherwise, the extension is treated as global.

Parameters
extensionNameThe name of the instance extension to check
layerNameNULL or the name of the layer to check the extension against
debuggerNULL or debugger to read configuration from when reporting debug messages
Returns
0 if the instance extension was INVALID
1 if the instance extension was VALID
