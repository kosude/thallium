int thvk_DestroyRenderSystem ( thvk_RenderSystem_t *  render_system)

This function frees the specified render system object. You should set the pointer to NULL after calling this function, as you would normally when calling free().

Parameters
render_systemPointer to the Vulkan render system to free.
Returns
Return status of the function. 0 indicates an error, and 1 indicates success.
Note
This function will always return 1.
