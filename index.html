Source code: kosude/thallium

Thallium is a planned C/C++ library that will provide the ability to write cross-renderer applications with the same interface. Cross-platform renderers (i.e. Vulkan and OpenGL) will be prioritised, but I plan to also implement Metal, DX11, and DX12 in the far future.
